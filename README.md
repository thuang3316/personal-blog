# personal-blog
Basic personal blog page hosted currently on localhost. Using postgresql to store blog data. 

Add basic authentication for access of editing and deleting blogs.

Set up form validation based on `express-validator`.

## Usage
Clone the repo. Run `npm install` to install all dependencies. Create an `.env` file to set up your database and provide login info. Run `npm start` to initialize the table.
Then you're good to go!
